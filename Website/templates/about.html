🏠 About House Price Predictor

House Price Predictor is a machine learning web application that helps estimate the prices of homes based on a wide range of features.

🔍 How It Works:

  • You upload a CSV file containing property data.
  • The data is cleaned and processed using a pre-trained model.
  • The app predicts house prices and returns a CSV with the results.

⚙️ Technologies Used:

  • Python, Pandas, NumPy
  • LightGBM for prediction
  • Sqlite3 for Databse
  • Flask, Jinja, JavaScript, HTML for the web interface
  • Bootstrap & CSS for styling
  • Jupyter Notebook for Machine Learning Models
  • Papermill for notebook execution
